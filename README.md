# Move.it
> Este App Auxilia Pessoas a Se Exercitar-se e se Movimentar de Tempos em tempos no Modelo Pomodoro


![](../header.png)

# 💻 Instalação

## Instale as dependências:

```yarn install``` ou ```npm install```

## Iniciando o Projeto:

`yarn dev` ou `npm run dev`
